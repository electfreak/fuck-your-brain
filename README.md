# Sandbox with JS problems

This project uses Vite + React + TypeScript.

### To run:

Ensure you use correct NodeJS version. If you use nvm:

```sh
nvm use
```

Run:

```sh
npm install
npm run dev
```

### Lint:

```sh
npm run lint
npm run format:check
```

### Formatting:

```sh
npm run format:fix
```
